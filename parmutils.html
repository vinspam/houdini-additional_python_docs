parmutils.decodeConditional(cond)¶

Decodes the conditional string into a list of groups containing lists of conditions.

The conditions consist of a list in the form [parm, op, value]

For example, { x != 0 } { y > 1 z 3 } decodes to:

[
[
[‘x’, ‘!=’, ‘0’]

], [

[‘y’, ‘>’, ‘1’], [‘z’, ‘==’, ‘3’]

]

]
