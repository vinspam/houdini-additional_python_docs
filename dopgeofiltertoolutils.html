dopgeofiltertoolutils module¶

dopgeofiltertoolutils.genericDopGeoForceTool(sceneviewer, nodetypename, validategeofunc, nodename, dop_prompt=None, force_prompt=None, exact_node_type=True)¶

This function is similar to the standard DOP filter tool, but is for DOP forces that use some external geometry to drive the force. As such they require an extra object level selection before creating the DOP node.
