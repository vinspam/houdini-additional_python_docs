dopwiretoolutils.setInitialPose(scriptargs, nodename=None)¶

Changes the initial pose of a Wire Object. This is done by creating a snapshot of the Wire Object’s current pose using a locked SOP and specifying the locked SOP in the Wire Object’s “initialpose” parameter. Any previous snapshot node used for the initial pose is removed.
